```bash

cd centrak-hub
php artisan serve

# Frontend
cd centrak-hub-ui/centrak-hub-ui-base
yarn install
yarn start
```

Open: **http://localhost:3000**

### 🔑 First Time Setup

1. Visit any page
2. Enter API Key (Bearer UUID from Laravel)
3. Click "Save Key"
4. Data will load automatically!
